# HTML & CSS To'liq Darslik

Bu loyiha HTML va CSS bo'yicha to'liq darslik saytini o'z ichiga oladi. Next.js framework'i yordamida qurilgan va ikki tilli (O'zbek/Rus) qo'llab-quvvatlaydi.

## O'rnatish

```bash
npm install
```

## Ishlatish

Development mode:
```bash
npm run dev
```

Production build:
```bash
npm run build
npm start
```

## Deployment

Bu loyiha Vercel, Netlify yoki boshqa Next.js hosting platformalarida osonlik bilan deploy qilinadi.

### Vercel'da deploy qilish:

1. GitHub'ga push qiling
2. Vercel'ga kirib, yangi project yarating
3. GitHub repository'ni tanlang
4. Avtomatik deploy bo'ladi

### Netlify'da deploy qilish:

1. `netlify.toml` fayli allaqachon mavjud
2. Netlify dashboard'da yangi site yarating
3. GitHub repository'ni ulang
4. Build command: `npm run build`
5. Publish directory: `.next`
